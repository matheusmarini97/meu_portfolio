# Pentaho - Queries e Jobs de Eventos

Esta pasta contém arquivos relacionados ao **Pentaho Data Integration (PDI)**, incluindo **transformações (Transformations) e jobs (Jobs)**, além de queries SQL utilizadas ao longo de diversos eventos.

## Conteúdo da pasta

- **Transformações (Transformations):** fluxos de processamento de dados que manipulam, integram e transformam informações para relatórios e métricas.
- **Jobs:** processos orquestradores que executam transformações, controlam a ordem de execução e garantem a consistência dos dados.
- **Queries SQL:** consultas desenvolvidas em **MySQL** para gerenciar inscrições, check-ins, atribuição de staff e geração de relatórios e certificados.

## Observações importantes

- Todos os **jobs, transformações e queries** foram utilizados **em produção** e refletem processos reais de eventos.
- Não se trata de material apenas para estudo; estes arquivos foram criados para suportar operações reais de gestão de eventos.
